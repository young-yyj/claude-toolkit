# 大纲模板

## 通用模板

以下模板适用于大多数 AI 领域知识点。根据知识点类型微调各章节侧重即可。

```markdown
# {知识点} 完整教程：从入门到精通

## 一、概述与背景
- 什么是 {知识点}
- 发展历史与重要里程碑
- 核心应用场景
- 与相关概念的关系

## 二、核心原理
- 基本概念与定义
- 工作机制详解
- 关键技术组件
- 与同类方案的对比

## 三、环境搭建与快速入门
- 前置知识要求
- 安装与配置
- Hello World 示例
- 基础 API 使用

## 四、基础用法
- 基本操作详解（含代码示例）
- 常见参数说明
- 输入输出格式
- 简单实战案例

## 五、进阶技巧
- 高级配置与调优
- 性能优化方法
- 常见陷阱与最佳实践
- 中级实战案例

## 六、高级应用
- 生产环境最佳实践
- 与其他技术的集成
- 扩展与自定义
- 高级实战项目

## 七、前沿进展与未来展望
- 最新研究成果
- 业界应用动态
- 未来发展方向
- 学习资源推荐

## 八、常见问题与踩坑经验
- 典型错误与解决方案
- 调试技巧
- 性能瓶颈排查

## 九、总结与参考资料
- 核心要点回顾
- 官方文档链接
- 重要论文
- 优质博客与教程
```

## 类型差异速查

| 知识点类型 | 章节调整要点 | 代码侧重 |
|-----------|-------------|---------|
| **算法/模型**（Transformer、RLHF、扩散模型等） | 第二章增加数学基础小节（线代/概率），增加"变体与改进"章节 | PyTorch/TensorFlow 复现、标准数据集验证 |
| **框架/工具**（LangChain、vLLM、LlamaIndex 等） | 第二章改为"核心架构与API"，强调类/接口关系，增加"生产部署"章节 | 安装配置、API 调用、自定义组件 |
| **概念/范式**（RAG、Agent、CoT、Prompt Engineering 等） | 第一章改为"概念解析"（定义+为什么+发展历程），增加"效果优化"章节 | 最简实现（≤50行）、完整流程搭建 |
| **基础设施/MLOps**（模型部署、量化、分布式训练等） | 第一章改为"背景与需求"（解决什么问题），增加"监控与运维"章节 | 配置文件、工具链选型对比、性能指标 |

## 章节组合建议

根据 SKILL.md Constraints 和 Step 4 的要求，生成时可灵活调整：

- **跨类型知识点**可融合多个类型的章节侧重
- 基础篇教程可减少第 六/七 章的深度
- 进阶篇教程可跳过第三、四章
- 所有类型均建议包含"常见问题与踩坑经验"和"总结"章节
